# Лабораторная работа 8

## Запуск приложения

1.  Убедитесь, что у вас установлен PHP.
2.  В командной строке перейдите в директорию `Task08`.
3.  Запустите встроенный веб-сервер PHP:

        php -S localhost:3000 -t public

    Если вы хотите, чтобы приложение было доступно **не только с этого
    компьютера**, а также с других устройств в локальной сети,
    запускайте так:

        php -S 0.0.0.0:3000 -t public

4.  Откройте браузер:

        http://localhost:3000

## Описание

Приложение «Сапёр» с бэкендом на PHP и базой данных SQLite.
