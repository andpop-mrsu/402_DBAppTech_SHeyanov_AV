# Лабораторная работа 9

## Запуск приложения

1.  Убедитесь, что у вас установлен PHP и Composer.
2.  В командной строке перейдите в директорию `Task09`.
3.  Установите зависимости (если они еще не установлены):

        composer install

4.  Запустите встроенный веб-сервер PHP:

        php -S localhost:3000 -t public

    Если вы хотите, чтобы приложение было доступно **не только с этого
    компьютера**, а также с других устройств в локальной сети,
    запускайте так:

        php -S 0.0.0.0:3000 -t public

5.  Откройте браузер:

        http://localhost:3000

## Описание

Приложение «Сапёр» с бэкендом на микрофреймворке Slim (PHP) и базой данных SQLite.
